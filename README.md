# Sistema de Gestión de Laboratorio Clínico

Proyecto desarrollado por Andy Martínez C.I: 84.491.009 para la materia de Backend (Ingeniería De La Programación).

## Tecnologías utilizadas
- **Node.js & Express** (Express Generator)
- **MySQL** (XAMPP)
- **JavaScript ES6** (Clases y Promesas)
- **Fetch API** para el Frontend

## Requisitos
1. Tener XAMPP instalado.
2. Importar la base de datos `laboratorio_clinico`.

## Instalación
```bash
npm install
npm start
